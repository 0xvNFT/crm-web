# Security Policy

## Supported Versions

| Version | Supported |
|---------|----------|
| latest (`main`) | ✅ |
| older branches | ❌ |

## Reporting a Vulnerability

If you discover a security vulnerability in **crm-web**, please **do not** open a public issue.

Instead, report it privately by emailing: **0xvNFT@gmail.com**

Please include:
- A clear description of the vulnerability
- Steps to reproduce
- Potential impact
- Any suggested fixes (optional)

You can expect an acknowledgment within **48 hours** and a resolution update within **7 days**.

> Note: The backend API is a separate private repository. If the vulnerability is related to the API/backend, please still report it via the same email.
